
LuminSkin

Packages And Dependancies 
    - Cypress
    - Gherkin
    - cypress-cucumber-preprocessor (Badeball)
    - cypress-esbuild-preprocessor (Bhamutov)

Installation Guide
    - Clone the Repository to your desired local machine with git clone <project-web-url>.
    - To install the dependencies, you can run npm ci or npm install

Execution

Run Test Locally
From the root of the folder, run npx cypress run for headless mode. You can run npx cypress open --e2e for browser mode where the test runner will be launched.
